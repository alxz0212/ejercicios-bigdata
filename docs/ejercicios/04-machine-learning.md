# Ejercicio 04: Machine Learning en Big Data

Aplicaremos técnicas de aprendizaje no supervisado para detectar patrones en conjuntos de datos complejos.

## Alcance
Este ejercicio se centra en dos familias de algoritmos:
- **Reducción de Dimensionalidad:** Principal Component Analysis (PCA).
- **Clustering:** K-Means y Hierarchical Clustering (HCA).

## Metodología
Utilizarás los scripts base proporcionados en `ejercicios/04_machine_learning/` para analizar el comportamiento institucional de los países (Dataset QoG).

### Tareas
1. **PCA:** Reducir las 100+ variables económicas a 2 componentes principales para visualización.
2. **Clustering:** Implementar K-Means para agrupar países. Deberás determinar el número óptimo de clusters ($K$) utilizando el método del codo o coeficiente de Silhouette.
3. **Interpretación:** Generar un perfil para cada cluster (ej: "Países con alta democracia pero bajo desarrollo").
