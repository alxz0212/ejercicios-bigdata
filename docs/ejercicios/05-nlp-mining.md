# Ejercicio 05: Minería de Texto y NLP

Introducción al procesamiento de datos no estructurados mediante técnicas de Natural Language Processing.

## Conceptos Clave
- **Tokenización:** Dividir texto en unidades procesables.
- **Stopwords:** Eliminación de ruido (artículos, preposiciones).
- **Similitud Jaccard:** Medición matemática de qué tan parecidos son dos documentos.
- **Sentiment Analysis:** Clasificación polar (Positivo/Negativo) basada en léxicos.

## Actividad Práctica
Encontrarás los scripts en `ejercicios/05_nlp_text_mining/`.

**Reto Principal:**
Implementar un sistema que compare descripciones de políticas públicas de diferentes países y detecte cuáles son semánticamente similares utilizando la Distancia de Jaccard.
