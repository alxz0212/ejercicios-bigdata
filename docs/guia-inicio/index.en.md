# Getting Started

Welcome to the getting started guide for the Big Data with Python course. This section will walk you through everything you need to begin working on the exercises.

## What you'll find here

### [Tools Installation](instalacion.md)

Learn how to install all the tools required for the course:

- Python 3.11+
- Git
- PyCharm (recommended IDE)
- Project dependencies

### [Your First Exercise](primer-ejercicio.md)

A step-by-step guide to complete your first exercise:

- How to navigate the repository
- How to run the exercises
- How to submit your work
- Best practices

### [Course Roadmap](roadmap.md)

Overview of all exercises and technologies:

- Learning levels
- Estimated time per exercise
- Technologies you'll master
- Recommended study plan

## Where to start?

!!! tip "First time with Git and Python?"
    Start with [Tools Installation](instalacion.md) where we explain step by step how to install everything you need.

!!! info "Already have everything installed?"
    Go straight to [Your First Exercise](primer-ejercicio.md) to start working.

!!! warning "Experienced developer?"
    Check the [Course Roadmap](roadmap.md) to see all exercises and choose where to begin.
